Jolidee Inc. (« nous », « notre », « nos ») exploite le site web https://majolidee.com/ (ci-après désigné par le terme « Service »). Sauf définition contraire, les termes utilisés dans la présente Politique de Confidentialité ont la même signification que dans nos Conditions Générales disponibles sur notre site.

Service : le site web https://majolidee.com/ exploité par Jolidee Inc.

Données à caractère personnel : désignent les données concernant un individu vivant identifiable.
